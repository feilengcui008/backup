鉴于有很多人不知道怎么用，这里给个详细点的使用教程。

1.你需要安装python和pip，并且用pip安装两个python库：

pip install beautifulsoup4

pip install requests

2.chrome浏览器登录douban网站，并在任何一个页面下拿cookie，如下

右键“查看”出现如下页面：

3.将复制的cookie写入cookie.txt，这个代码库中cookie.txt中默认是空的

4.执行python douban.py --username="你的豆瓣用户名" --login
